Zoldhum

Fejlesztő: Sallai András

Továbbfejlesztés

git clone https://github.com/andteki/zoldhum.git
cd zoldhum
composer install
cp .env.example .env
...

Adatbázis

Hozzuk létre a database.sqlite fájlt: 
touch database/database.sqlite
Szerkesszük a .env állományt: 
DB_CONNECTION=sqlite
# DB_HOST=127.0.0.1
# DB_PORT=3306
# DB_DATABASE=laravel
# DB_USERNAME=root
# DB_PASSWORD=
Végezzük el a migrálást: 
php artisan migrate
php artisan key:generate

Végpontok

végpont metódus azonosítás
/api/register post nem új felhasználó
/api/login post nem belépés
/api/positions get nem beosztások lekérdezése
/api/positions post nem új beosztás
/api/positions/{id} put nem beosztás frissítése
/api/positions/{id} delete nem beosztás törlése
/api/employees get nem dolgozók lekérdezése
/api/employees post igen új dolgozó
/api/employees/{id} put igen dolgozó frissítése
/api/employees/{id} delete igen dolgozó törlése
